## Markdown Syntax

### Horizontal Slides

Use `---` to create horizontal slides

```markdown
# Slide 1
---
# Slide 2
```

-----

### Vertical Slides

Use `-----` (5 dashes) for vertical slides

```markdown
# Main Slide
-----
## Nested Slide 1
-----
## Nested Slide 2
```
